MATH 1650: Mathematics for Computing Science

This course surveys several mathematical concepts used in Computing Science. Some topics include logic; circuits; number systems; finite state machines; vector and matrix algebra; systems of linear equations; and linear transformations.

Pre-Requirements

None.
